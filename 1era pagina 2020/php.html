PHP

El preprocesador de hipertexto, es un lenguaje de scripting escrito en Perl y C. Se utiliza, principalmente, en la programación de páginas webyaplicaciones web dinámicas. PHP es considerado como un lenguaje de programación web apto para inexpertos y es compatible con HTML. Por estas razones, suele ser uno de los primeros lenguajes que aprenden los futuros programadores. A pesar de que hay quienes consideran que PHP es un lenguaje desactualizado, muchos propietarios de páginas web siguen dependiendo de él hoy en día. Entre las principales ventajas que ofrece, se incluye el hecho de que es un lenguaje con soporte de base de datos y una integración eficiente con el protocolo de internet. . Se trata de un lenguaje con licencia de código abierto y disponible de forma gratuita.

PHP procesa el código del lado del servidor evitando así, la interpretación por parte del navegador, como ocurre en el caso de otros lenguajes de programación web muy conocidos. Por ello, PHP está incorporado en HTML (el cual no contiene información de estado) y provoca una mayor carga en el servidor que otros lenguajes de programación que solo transfieren el código fuente a un navegador web dedicado.
